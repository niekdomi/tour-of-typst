# Listen

Jedes Experiment braucht eine Materialliste und eine Vorgehensweise. Unsere umfasst Brot, Butter,
eine Hochgeschwindigkeitskamera und eine besorgniserregende Gleichgültigkeit gegenüber
Küchenhygiene. Typst kennt zwei Listentypen:

**Ungeordnete Listen** verwenden `-`:

```typst
- Weissbrot
- Gesalzene Butter
  - Streichfähig, nur auf einer Seite aufgetragen
```

Das wird so dargestellt:

- Weissbrot
- Gesalzene Butter
  - Streichfähig, nur auf einer Seite aufgetragen

**Nummerierte Listen** verwenden `+`:

```typst
+ Eine Scheibe mit Butter auf der Oberseite vorbereiten
+ Auf Tischhöhe halten
+ Ohne Drall fallen lassen
+ Den Fall aufzeichnen
+ Das Ergebnis beweinen
```

Das wird so dargestellt:

1. Eine Scheibe mit Butter auf der Oberseite vorbereiten
2. Auf Tischhöhe halten
3. Ohne Drall fallen lassen
4. Den Fall aufzeichnen
5. Das Ergebnis beweinen

> [!TIP] Zwei Leerzeichen Einzug erstellen verschachtelte Listen. So können wir Unterpunkte wie
> Brotdicke und Buttermarke angeben.

## Deine Aufgabe

Fülle den Abschnitt **Methoden** mit zwei Listen aus:

1. Eine **ungeordnete Liste** der Materialien: Brotspezifikationen, Butterdetails und Ausrüstung
   (Kamera, Tisch, Bodenschutz)
2. Eine **nummerierte Liste** mit der Beschreibung des Versuchsablaufs Schritt für Schritt

Die Gutachter müssen unsere Arbeit replizieren können. Die Wissenschaft verlangt es.
